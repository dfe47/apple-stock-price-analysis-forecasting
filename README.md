# Apple Stock Price Analysis & Forecasting

## Overview

Time series analysis project focused on Apple stock data to identify market trends, analyze trading activity, and forecast future stock prices using statistical modeling techniques.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels (ARIMA)

## Key Insights

- Analyzed 1,255 Apple stock records
- Evaluated stock price trends and trading volume patterns
- Built 30-day and 90-day moving average indicators
- Performed correlation analysis on market variables
- Developed an ARIMA model to forecast stock prices over a 30-day horizon
